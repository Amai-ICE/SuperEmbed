# Super Embed

Super EmbedはX等のリンクをDiscord用の見やすい埋込みに変換するBOTです。

## 参考画像

![img.png](.github/resources/img.png)

## 使い方

### 要件

1. java 21

config-example.tomlを参考に、config.tomlを作成し、
tokenにBOTのトークンを入れて`java -jar SuperEmbed-$VERSION$.jar`で起動してください。

# 増やしたい機能

- Xのプロフィール
- Reddit
- Qiita
- Pixiv
- Misskey
